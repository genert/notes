# Ruby

## Requirements

Install [Ruby](https://www.ruby-lang.org/en/).

## History

Yukihiro Matsumoto created Ruby in 1993. As a language, Ruby is interpreted, object-oriented, dynamically typed language from a family of so-called scripting languages.
Interpreted means that types are bound at execution time rather than compile time.
Object-oriented means the language supports encapsulation, inheritance through classes, and polymorphism.
